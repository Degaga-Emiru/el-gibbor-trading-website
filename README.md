# El Gibbor Trading One Member PLC Website

A modern, responsive, and professional corporate website for **El Gibbor Trading One Member PLC**. The website showcases the company's import and export services, products, projects, and corporate information while delivering a seamless user experience across all devices.

---

## About the Company

**El Gibbor Trading One Member PLC** is an Ethiopian trading company specializing in:

- Import & Export
- Tyre Supply
- Automobile Trading
- Automotive Spare Parts
- General Trading
- Business Supply Solutions

The website is designed to strengthen the company's online presence by providing customers and partners with comprehensive information about its products, services, and business operations.

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/el-gibbor-trading-website.git
```

Navigate into the project

```bash
cd el-gibbor-trading-website
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
